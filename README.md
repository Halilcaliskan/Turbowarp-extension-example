# Türkçe
# Merhaba TurboWarp Extension

Basit bir TurboWarp extension.
Tek bir dikdörtgen (stacked) blok ekler.

## Blok
- merhaba yaz → console'a "Merhaba Dünya!" yazar

## Kullanım
1. https://turbowarp.org/editor aç
2. Extensions → Custom Extension
3. https://github.com/Halilcaliskan/Turbowarp-extension-example/blob/main/README.md?plain=1 Linkini yapıştır
4. Unsandboxed şekilde tamamla

# English

# Hello TurboWarp Extension

A simple TurboWarp extension.
Adds a single rectangular (stacked) block.

## Block
- Type hello → prints "Hello World!" to console

## Usage
1. Open https://turbowarp.org/editor
2. Extensions → Custom Extension
3. Paste the link: https://github.com/Halilcaliskan/Turbowarp-extension-example/blob/main/README.md?plain=1
4. Complete in unsandboxed mode
